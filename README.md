# Backend Chronicle

## Pre requisites

- Docker and docker-compose
- Node JS
- NPM

## Getting Started

```sh
  git clone <repo>
  cd <repo>
  npm i
```

Ir em `docker` e 

```sh
  docker-compose up -d
```

```sh
  npm run dev
```

- pgAdmin em localhost:81
- Banco PostgreSQL em localhost:5432
- API em localhost:8080
